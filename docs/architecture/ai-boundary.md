# AI Boundary — LLM Adapter Architecture

The LLM Adapter provides a secure, deterministic bridge between the MS Access MCP server and external Language Models. By isolating provider-specific SDKs into pluggable adapters, the core server maintains its deterministic nature while exposing advanced AI capabilities safely.

> **SDD Reference**: See `sdd/llm-adapter/design` for full architecture, data flow diagrams, and security model.
